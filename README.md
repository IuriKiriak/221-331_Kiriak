# Лабораторная работа: Менеджер учётных данных

В рамках данной работы реализовано приложение с окном аутентификации и окном отображения учётных данных. Пользователь вводит пин-код (мастер-пароль) для разблокировки хранилища. При корректном вводе выполняется расшифровка зашифрованного JSON-файла с использованием AES-256 и библиотеки OpenSSL, после чего данные загружаются в память и отображаются в таблице.

Файл содержит не менее 10 записей и имеет размер более 2 КБ. Все пароли отображаются в маскированном виде. Реализована фильтрация по URL сайта. Интерфейс адаптивный и корректно масштабируется.

Добавлена защита: данные в файле зашифрованы, ключ генерируется из пин-кода через хеш-функцию. Также реализован второй слой шифрования для логинов и паролей в памяти. Расшифровка выполняется только по запросу пользователя.

При обнаружении ошибок (неверный пин-код, повреждённый JSON) отображается сообщение об ошибке. В случае подозрения на атаку ввод блокируется.
